1. List all the tools yo might use as a web developer.
Ans : As a web developer, you'll likely use a variety of tools to design, develop, test, and deploy websites. Here's a comprehensive list of tools that are commonly used:
Text Editors/IDEs: Visual Studio Code, Sublime Text, Atom
Version Control: Git, GitHub, GitLab
Web Browsers: Chrome, Firefox, Safari, Edge
Front-End: HTML/CSS preprocessors, JavaScript frameworks (React, Angular, Vue.js)
Back-End: Node.js, Python, Ruby, PHP, MySQL, MongoDB
API Testing: Postman, Insomnia
Package Managers: npm, Yarn
Task Runners/Build Tools: Webpack, Gulp, Grunt
Testing/Debugging: Jest, Selenium, ESLint
CMS: WordPress, Drupal, Joomla
Deployment/Hosting: Heroku, AWS, Netlify, GitHub Pages
Design Tools: Adobe XD, Sketch, Figma
Collaboration: Slack, Trello, Jira
